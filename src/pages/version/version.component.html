Linux From Scratch

Linux From Scratch est un livre décrivant l'installation étape par étape d'un système d'exploitation Linux à partir du code source.

  • ✅ Comprendre les interactions entre les différents programmes constituants la base d'un système Linux
  • ✅ Connaître la base de la configuration d'un système Linux
  • ✅ Réaliser votre propre distribution Linux
